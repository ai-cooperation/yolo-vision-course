場景示範：標籤前 vs 標籤後

挑幾個代表性現象，看同一條流程怎麼應對。左邊原始、右邊 YOLO 標註， 注意框、類別、信心度與追蹤 ID 的變化。所有標註皆由本課程的 Colab 流程實際跑出。

偵測：三個經典現象

分割：從「框」到「物體邊緣」

偵測給的是方框；分割更進一步，畫出物體的精確輪廓（遮罩），到像素級。 同一條流程換成 -seg 模型就能做——適合量面積、去背、看瑕疵形狀。

左原始、右分割遮罩。注意它不只框住物體，而是沿著輪廓著色——這是偵測與分割最直觀的差別。
