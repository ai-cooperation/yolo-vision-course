第 5 章

實務操作：在 Colab 上跑通

免安裝、用免費 T4 GPU、檔案放 Google Drive，不佔本機。從標準標的入門， 一路做到自己的模型與邊緣部署。

三個級別：從標準到專用

標準級

標的：人/車/動物（COCO 內建）。
來源：YouTube／公開影片。
訓練：免，直接用。

通用級

標的：居家的人/車/寵物。
來源：自家 ipcam（RTSP）。
訓練：免，仍用 COCO。

工業級

標的：自家產品（COCO 沒有）。
來源：工廠 RTSP。
訓練：要——自己標註＋合成資料微調，再做產線計數。

循序操作步驟

  1. 開 Colab、選 T4 GPU：開啟下方 Notebook，執行階段 → 變更執行階段類型 → T4 GPU。
  2. 設一支公開影片網址：把參數格的 YouTube 連結換成你要測的（街景、人車最容易看出效果）。
  3. Run All（標準級）：自動下載 → 偵測 → 追蹤 → 印出辨識報告與標註影片。對照第 2 章看懂四個標籤部分。
  4. 換成你自己的標的：要偵測 COCO 沒有的東西（某零件、某產品），用 Roboflow 標 50–100 張。
  5. 在 Colab T4 訓練（工業級）：用標好的資料微調，得到自己的 best.pt。
  6. 用新影片驗證：在沒看過的影片上跑，確認泛化、不是死記。
  7. 部署到邊緣：匯出 NCNN，放到 Raspberry Pi 5 現場推論／計數。

訓練自己的模型（工業級）

當標的是 COCO 沒有的東西，流程是：標註 → 合成補強 → Colab T4 微調 → 新影片驗證 → 匯出邊緣格式。 小樣本時用 Copy-Paste 合成（真實產品圖貼到真實背景）能大幅省下標註量。驗收看 mAP 與在「沒看過的影片」上的表現。

部署到邊緣

訓練在雲端（GPU），推論在現場（省錢、低延遲、斷網可用）。把模型 export(format='ncnn') 後放到 Raspberry Pi 5，接上 RTSP 相機，設好 ROI 與計數線，就能現場即時計數並回傳數量。
